TEM蛋白质负染制样与观察

Posted on 2021-04-15 Edited on 2021-04-21 In 技术 Views:
Symbols count in article: 3.6k Reading time ≈ 3 mins.

蛋白负染TEM观察

负染制样

1. 准备蛋白样品

用自己的缓冲液稀释蛋白至$200-300nM$，一个样品需要$4 \mu L$蛋白。按照一个颗粒的大小计算浓度，缓冲液中最好不带有甘油。

2. 选择碳膜

小的蛋白最好选择超薄碳膜，这样衬度会高一些，但是收到的数据较少。

大的蛋白用普通碳膜和超薄碳膜的效果都差不多，但是普通碳膜收数据会快一些，能用的照片也会多一些。

3. 辉光

准备一个铺有一层滤纸的小玻璃皿（平台有），用高精准直头镊子（平台提供）将铜网盒中的铜网夹出，放在小玻璃皿上，注意保持黑灰色的碳膜面朝上，亮紫红色的铜网面朝下。

将小玻璃皿放入辉光仪中，关上辉光仪的腔门，使阀门的长端水平向右。之后按照下列程序进行辉光：

  1. 抽真空：打开真空泵开关，用手按住门以防止漏气，抽真空耗时$3min$，使内部气压为$0.37mBar$，保持$10s$。
  2. 辉光放电：将旋钮从“OFF”拧到“MED”，在15mA的条件下辉光$30s$，此时应该从侧窗看到仪器内腔中有紫色辉光出现。（刚买的仪器可以电流小一些，时间短一些，主要取决于铜网的承受能力，超出其承受范围铜网会破）
  3. 辉光结束：将旋钮从“MED”拧到“OFF”，之后关闭真空泵开关，用手指堵住放气孔，逆时针缓慢旋转门上的阀门，感受到吸力时间歇性缓慢放气，直到腔内气压与外界相同为止。打开腔门，取出小玻璃皿。

4. 蛋白附着

  1. 将盛有铜网的小玻璃皿放在实验台上，用高精准自锁直头镊子夹起铜网，镊子保持水平，夹起铜网的一点点边，之后将夹子锁死，轻轻晃动夹子铜网不动。将夹子水平放置在架子上，保持铜网的碳膜面朝上。
  2. 使用1-10$\mu L$移液枪吸取准备好的蛋白样品，垂直滴在铜网上，使蛋白附着$30-60s$。（具体时间可根据实际情况调整，天气干燥时可以缩短时长）

5. 负染

在制样前就取出乙酸铀，若发现沉淀较多则首先离心。

  1. 在蛋白附着的同时取一片封口膜和滤纸。用10-100$\mu L$移液枪吸取离心后的$2\%（w/v）$乙酸铀$100\mu L$，平均分成三滴点在封口膜上。[注意放射性，枪尖收在特定袋子中带走]
  2. 蛋白附着完成后，左手持滤纸片，右手持镊子，将铜网垂直与滤纸片接触，待观察到铜网上最后一点液体被吸走之后立即将铜网的碳膜面放在第一滴乙酸铀上，来回摇动$10s$，之后马上放到第二滴上摇动$10s$，最后在第三滴上摇动$10s$。之后将镊子重新水平放回架子上，保持碳膜面朝上，放置$1-2min$。此时铜网上应保存有乙酸铀液滴。
  3. 将铜网垂直于滤纸片，将铜网上的液体吸干之后将夹子水平放置在架子上，晾干$2min$。
  4. 将滤纸片覆盖在封口膜上，一起扔在袋子中，走的时候带走。

6. 样品保存

自带铜网保存盒，将镊子竖起来，解自锁之后慢慢将镊子放入小空中，放入之后放开镊子，使铜网落入空中。如果没有完全进孔就用镊子轻轻推一下，之后慢慢将盒子盖上，以防静电。

在记录卡上记录蛋白样品的信息，格式为“蛋白名称XXX-制样时期XXXX-制样人XXX”，如：degQ-20210412-CWH

自带仪器和物品 实验室准备
1-10$\mu L $移液枪和枪尖 铜网
10-100$\mu L$移液枪和枪尖 高精度自锁镊子
记号笔，签字笔 2%乙酸铀（保存在冰箱的铅盒中）
铜网盒及标签纸
EP管架
计时器
蛋白质溶液（保存在冰盒上）

TEM上机观察

使用前检查

  1. TEM配备一台透射电子显微镜，CCD照相机，两个控制台，两台显示器（左边显示器1连接电子显微镜设备，右边显示器2显示图像处理软件）
  2. 左边显示器应打开两个软件，分别为“User interface”和“Imaging & Analyses”
  3. 将镜筒上伸出的铜丝浸泡在液氮中：先在保温桶中倒入半桶液氮，将铜丝放入。之后再补充到满，补充的时候注意用盖子盖住铅玻璃台面，否则容易崩裂GG
  4. User interface界面左边Setup界面显示三片黄色。

上样

平台TEM共配备三台上样杆，分别对应于普通观察样品，冷冻样品和断层扫描三维重构样品。蛋白质负染样品使用普通观察样品杆即可。

  1. 首先应记录上样的样品信息。
  2. 使用上样杆架子上的金属针将杆前部的压片抬起。使用弯头镊子夹起铜网的一点点边，轻轻放入杆前面的凹槽中。再使用金属针轻轻将压片放下，可见铜网被夹在上样杆的前部。
  3. 将上样杆从保护筒中取出，使用洗耳球吹前部和杆身，除去小的杂质。注意不能用手碰杆。
  4. 上样前检查：
    • 显示器1右下角的位置坐标是否清零，如果不为零在Search界面上的拓展按钮中的holder按键将其清零。
    • 上样位置的红灯不亮
    • Setup界面显示Col Valves Closed为黄色，真空度窗口前三个均为绿色（从上至下分别为1，6，9）。
  5. 将上样杆上突起处朝向close线，对准后将上样杆推进去（如果遇到阻力可稍微转动），推动过程中注意真空度检测窗口的column值的大小应保持不变或轻微浮动，若发生较大调动应停止推进待恢复后继续上样。
  6. 将上样杆推动至杆身黑色刻度处时停止，此时上样处红灯亮起，镜筒开始抽真空，保持$3min$。此时在显示器1下边选择Single Tilt单擎杆选项，点击回车键。
  7. $3min$后，红灯灭，右手逆时针旋转上样杆，同时左手从反方向抵住，边旋转边缓慢推进，感受到吸力后仍继续缓慢旋转，转至Open线后位置正确，上样杆上样完成。

像的寻找和观察

  1. 在Setup界面选择Satus Col Valves colsers，将其点成灰色状态即为打开。之后可观察到在显示器1右边的示意图中阀门打开。
  2. 检查物镜光栅是否打开（在左边），如果没有打开将其打开（开关旋到左边）。
---------------------------以下步骤都通过铅玻璃屏观察---------------------------
  1. 调节控制台2的Magnification旋钮，将放大倍数调节到100-500X左右，同时调节控制台1的Intensity和滚动球，保持光圈充满屏幕。
  2. 找到一个较干净的不破的目孔，调节控制台2的Magnification旋钮将放大倍数放大至4000-5000X，确定要观察的对象并将其调整到视野中央。
  3. 按下控制台2上的Eucention focus键（目的是使显微镜状态处于合轴的最佳状态），退出物镜光栅，按控制台2上的z axis↑、↓按键，将显示器1右下角的“z axis”调节到$100\mu m$左右。
  4. 关注已经找到的要观察的对象，上下按控制台2上的z axis↑、↓按键，调整到该像最虚的条件下（即再上下移动后物象均变的清晰），此时的z axis即为要确定的值。为了确定该值是否可靠，可以在Search的拓展选项中点击Set alpha，将样品旋转$15^{\circ}$，再点击Set alpha使其旋转回来，观察转回来的时候观察对象在视野中的位置是否发生改变，如果不变则说明z axis设置完成。之后在同一目中进行观察时，都不需要再设置z axis。
  5. 将物镜光栅调回，将放大倍数放大到50000-60000X，准备获取图像。（刚开始可以从较低倍数开始，逐渐放大到135000倍。）

图像获取

---------------------------以下步骤都在显示器观察---------------------------
  1. 首先按控制台2的R1键将显示屏抬起，之后选择显示器中左边CCD/TV camera，进入图像获取界面。
  2. 点击Search，获取当前观察到的实时图像。图像将显示在显示器2上。
  3. 点击显示器2右边栏FFT图标进行傅里叶变换，在显示器2上将出现一个显示波形图的分屏，此时通过转动控制台2的Focus下层旋钮将显示器1右下侧的focus step设置为3，之后转动Focus上层旋钮，将傅里叶波形图外圈的波调没，此时处于正焦位。仍存在外围波形则为欠焦，外围发白则为过焦。在同一目中观察时只需要进行一次傅里叶变换。
  4. 按控制台2的R2键将Defocus选项归零，之后旋转Focus上层旋钮，将Defocus调整为-2.03$\mu m$左右。
  5. 点击显示器1左侧CCD/TV camera界面的Acquire，获取图像，即在显示器2上显示获取的图像。
  6. 仍然选择Search，将傅里叶变换窗口关闭，摇动控制台2的转动杆，可以左右移动视野寻找较好的视野进行照片获取。
  7. 获取的图像都展示在显示器2的底边栏，若要保存为tif格式，需要在图像上右键选择export data，选择位置保存；若要保存为emi格式，则需在显示器2右边栏点击保存按钮，选择位置保存。

下样

  1. 观察完毕，下样前检查：
    • 关闭Setup界面中的Col Valves closers（使其变为黄色状态）。
    • 将物镜光栅去下（旋回右边）。
    • 在Search界面上的拓展按钮中的holder按键将坐标位置清零。
  2. 用手扶着上样杆，向外缓慢拉，并且缓慢旋转。上样杆卸下后放回原处，用金属针卸下铜网放回铜网盒。
