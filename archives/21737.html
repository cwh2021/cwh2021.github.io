PYMOL开源版的安装

Posted on 2021-01-27 Edited on 2021-01-29 In 生物学软件 Views:
Symbols count in article: 2.1k Reading time ≈ 2 mins.

PYMOL: A molecular visualization system on an open source foundation, maintained and distributed by Schrödinger.

PyMol是一款开放源码，由用户赞助的分子三维结构查看软件，适用于创作高质量的小分子或生物大分子（特别是蛋白质）的三维结构图象，由Schrödinger开发和维护。软件以PY+MOL命名，”PY”表示它是由计算机语言Python所编写的，”MOL”表示他是用于显示分子(molecule)结构的软件。

PYMOL目前提供有教学版，企业版，政府和学术版和开源版。其中只有教学版和开源版可以免费下载。不同版本的在功能上的区别如下图所示（具体比较见PYMOL官网）。

之前申请过教学版，寒假想试一下开源版，记录一下安装的过程。

2. 下载所需文件

访问加州大学Unofficial Windows Binaries for Python Extension Packages网站，搜索Pymol-open-sourse，下载对应于系统和Python版本的如下文件：

pymol_launcher‑2.1‑cp39‑cp39‑win_amd64.whl

pymol‑2.5.0a0‑cp39‑cp39‑win_amd64.whl

Pmw‑2.0.1‑py3‑none‑any.whl

搜索Numpy+MKL，下载对应于系统和Python版本的如下文件：

numpy‑1.19.5+mkl‑cp39‑cp39‑win_amd64.whl

注意：一定要记住自己的安装路径！可以先创建一个文件夹将以上文件都放进去。
